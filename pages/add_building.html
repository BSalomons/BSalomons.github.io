Add a Building

You have the option of going to the Building object to create and populate the fields but the form for the Building is embedded within the Property object’s form. This is the easiest way to add and edit these fields.

Navigate to the Property you want to add a Building and scroll down to the Building section to complete the fields.

  • A generic name is all that is needed here as most times there is only one Building on a Property.
  • If there are multiple buildings, pick some method of differentiating them
  • This is an old field and isn’t really needed
  • Provide a basic description of the building
  • Must complete this field as this number is used to calculate the Property’s total rentable area
  • This is the total rentable area for the entire Building, not just what is available
  • Must complete this field as the number is used to calculate the Property’s total footprint area and then this number is used to calculate the site coverage ratio
  • Depending on how a building is measured (usually office buildings) then this number might differ from the Rentable Area. Most times they are the same number
  • Old field that doesn’t need to be completed

At this point, click the “Save and Continue Editing” button on the right-hand side before proceeding.

Once the save button is hit, the Building record is created and the Database assigns a unique key to it. This number is then displayed in the heading and can be used to easily assign Premises to it.

There are additional fields within the Building object that are not displayed in the Property form. In the header for the Building, there is a “edit” button and if pressed it will go to that Building record’s form. From here, you will then see the unique key for the associated Property record. The fields that exist here may be used in future editions.

Once the Building record is created, a new form below for a second building is displayed. If it is blank then there is no record, it is just there to input for a second building if need be. There is also a button underneath that says “Add another Building” in case you wanted to add all the Buildings prior to hitting save. Best practice is to save often.
