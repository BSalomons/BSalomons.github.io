In Contract - Unconditional Offer for Sale or Lease

Once conditions have been waived and satisfied for a deal transaction, the following steps must be followed immediately.

Click Here to jump down to the Lease section

Sale

Prerequisites:

In the unlikely event you happen to write an unconditional offer that is accepted by the seller, follow the steps for a Pending Deal except the Status field will be marked as “In Contract” and you won’t need to enter the conditions.

At this stage, it is very important to check that the data entered while it was “Pending” is still accurate.

Updating a Pending Offer to In Contract:

Conditions have been waived and the transaction is proceeding to closing. The In Contract record must be double checked that applicable deal terms are still accurate. Amendments or other changes may have occurred to the deal subsequent to the initial acceptance.

The following fields must be checked and updated by the agent once an “In Contract” deal occurs:

  • Change the Status from “Pending” to “In Contract”
  • When the save button is hit, emails will be sent out to the broker and admin staff
  • When the save button is hit, all the conditions that were listed will be marked as complete and, if the condition removal date was not entered, it will be marked with todays date.
  • Make sure this price is still accurate as it may have changed from an amendment
  • Make sure this date is entered correctly
  • If unknown, use your best estimate or worst-case scenario
  • Since deposits and correspondence will be sent to both lawyers and the brokers, make sure their addresses and contact information is accurate. The fields below the party’s names is what is associated with that person. If blank, then nothing is inputted.
  • Make sure this is still accurate.

At this point, click the “Save” button that will be hovering at the bottom of the page before proceeding.

The remainder of the record should be updated by the agent to ensure its accuracy. The following explains the various “Related Lists” associated with the In Contract record.

  • Make sure the Date Removed is correct
  • Make sure the Complete button is checked
  • Make sure this is accurate with the correct allocations

Lease

In the unlikely event you happen to write an unconditional offer that is accepted by the landlord, follow the steps for a Pending Deal except the Status field will be marked as “In Contract” and you won’t need to enter the conditions.

  • Ensure the lease value is still accurate.
  • To calculate, there is an excel spreadsheet that can be used to calculate:
    • >3. General Office>Templates – Office>Deposit Calculator
  • Make sure this date is entered correctly as it may have changed since condition were waived
  • It may not yet be known but it is very important to update this when it is known. Diarize it and set a reminder to come back and update it when known.
  • Since deposits and correspondence will be sent to both landlords and the brokers, make sure their addresses and contact information is accurate. The fields below the party’s names is what is associated with that person. If blank, then nothing is inputted.
  • Make sure this is still accurate.
  • This can be calculated beside the Lease Value
