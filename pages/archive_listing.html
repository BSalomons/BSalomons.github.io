Archive Listing

Once a listing has expired, or has been sold or leased, the following steps are taken to archive it from the Database, Salesforce, website and our server.

  1. Navigate to the applicable record on the Database by following one of the methods in Checking if Property or Listing Exists in Database.
  2. Uncheck the Active field, or input a date in the Is Active Expiry which will uncheck the Active field when the date condition is met.
  3. Click Save
    1. At this point it will be removed from the frontend of the website but still searchable in the Database
  4. Archive the Listing folder on the K Drive. This means “cutting” (Ctrl+X) and “pasting” (Ctrl+V) the folder under the Active directory (1. Listings>Active) to the Past Active Listings directory (1. Listings>Archived>Past Active Listings).
