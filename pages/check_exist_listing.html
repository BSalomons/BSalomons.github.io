Check if Listing Exists

There are several methods to determine if the Property or Listing already exists in the Database. It might be under a different name or the information has not been updated yet. If the Property or Listing resides in Red Deer, Red Deer County, Blackfalds or Sylvan Lake, there is a good chance it already exists in the Database. Here is how you check:

  1. Map Method
    1. Another method will have to be used if the Property is a condo.
    2. On the left hand side, click “Map”
    3. A map showing all Listings, Properties and shapes will load. By default, only the Listings will be toggled.
    4. Click the “shapes” button and the Properties that are in the Database will load as yellow polygons.
    5. Click the shape and the Property information will be displayed which means it exists. Edit the information if it is not current. If a Listing exists for the Property, it will also be displayed here and can click to edit.
    6. If the Property does not have a yellow polygon, then it may not be in the Database. This would be true if the property was subdivided since 2019.
  2. Address Check
    1. Click the Property of the Listing link on the left-hand side
    2. In the “Keyword” box, enter the address of the Property or the Listing.
    3. The address must be entered exact so if nothing shows then attempt some variations (such as no hyphen) or only enter the start of the address (ex. 17 Queens or 9503)
    4. Try not to use hyphens. Portions of words will work so “Queens” will display the Property or Listing if the actual is “Queensgate”
  3. LINC Number
    1. Enter the 10 digit number
    2. Can be a portion of the number.
