Add a New Company

It is straight forward to create a new Company. While not required, it is best practice to create a Company before you add a Contact to that Company. This is because a Company is a parent record to a Contact and therefore some of the information from the Company will be inherited by the Contact record. If the reverse order is done, then this doesn’t happen.

  • Use the full name of the company if possible.
  • If the company is in multiple locations, add the city name. For example, CBRE Calgary and CBRE Edmonton.
  • Not required but add if you want
  • Choose the appropriate selection
  • General phone line
  • If needed
  • Use the full URL as it will be a link on save
  • Dependent field based on the selection from Type above
  • Choose all appropriate selections
  • Pick the most applicable industry. Other can be used
  • Add anything that may need further explanation
  • There is a search bar where you start typing the address and make the appropriate selection. Once selected, Salesforce will populate the street, city, Province, postal code and country
  • Double check the postal code, sometimes it gets it wrong
